Bem-vindo ao Sistema de Whitelist

Para acessar o formulário, você precisa usar o comando /whitelist no Discord

Para solicitar sua whitelist, siga estes passos:

  1. 1 Entre no nosso servidor Discord
  2. 2 Use o comando /whitelist no canal apropriado
  3. 3 Clique no botão que aparecerá na mensagem
  4. 4 Preencha o formulário com suas informações
